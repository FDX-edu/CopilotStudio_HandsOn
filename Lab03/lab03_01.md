# 날씨 조회 하기 

1. **날씨**라는 이름으로 새로운 토픽을 만들고, 트리거 문구를 아래와 같이 입력합니다.
    ```
    오늘 날씨 어때?
    이번 주 날씨 알려줘.
    [지역명]의 현재 날씨는?
    내일 비 올까?
    [지역명]의 기온이 몇 도야?
    ```
    ![image](https://github.com/user-attachments/assets/b3f3e305-b946-4af6-ae6e-4bc244f08192)


2. **질문하기** 노드를 추가하고, **어느 지역의 날씨를 알려드릴까요?** 라는 질문 문구 및 "구/군/시" 엔터티 형식의 Topic.varLocation 변수를 설정합니다.
    ```
    어느 지역의 날씨를 알려드릴까요?
    ```
    ```
    varLocation
    ```

    ![image](https://github.com/user-attachments/assets/18cedde5-e593-4bbf-abcd-8b625c29a653)


3. **+**를 클릭하고, **작업 추가** - **새 Power Automate 흐름**을 클릭합니다. </br>
   
   ![image](https://github.com/user-attachments/assets/3ee5d3e0-98b6-4dce-80fa-e57dee74baae)

    브라우저의 새로운 탭 또는 윈도우에 PowerAutomate 화면이 열립니다.

    ![image](https://github.com/user-attachments/assets/fe55d5a5-714a-4fdd-9954-f3427ce17285)




4. 왼쪽 상단에서 흐름 이름을 **날씨조회** 로 변경하고, **Copilot에서 흐름 실행**을 클릭 후 오른쪽 창에서 **+ 입력 추가**를 클릭하여 **텍스트**를 선택한 후 입력 부분에 **지역**이라고 입력합니다.

    ![image](https://github.com/user-attachments/assets/d3b7033e-ab97-4f37-804c-9c684b950bd9)


    ![image](https://github.com/user-attachments/assets/7639d707-2798-4b43-a08f-51582cde34e5)


5. **Copilot에서 흐름 실행** 아래의 **+**를 클릭하여, 왼쪽에 작업 추가 창이 열리면 커넥터 유형에서 **표준**을 선택합니다.</br>
   **MSN 날씨**를 선택합니다. 

    ![image](https://github.com/user-attachments/assets/64e5f395-706d-497e-8cac-90e4e9a6cf4e)

    **오늘 예보 가져오기**를 선택합니다.

    ![image](https://github.com/user-attachments/assets/9a189361-d4d7-44b5-9bce-c840b59bcefa)

    **오늘 예보 가져오기**가 추가되었습니다.
   
    ![image](https://github.com/user-attachments/assets/974f3ff2-206a-44c0-b945-0dc90a6723d7)

6. **오늘 예보 추가하기**를 클릭하고, 오른쪽에서 **위치** 아래 입력창에 마우스로 클릭하면, 입력창 오른쪽에 번개 아이콘과 fx 아이콘이 보입니다.</br>
   번개 아이콘을 클릭하면 오른쪽에 사용 가능한 변수들이 보입니다.</br>
   Copilot에서 흐름 실행 아래의 **지역**을 선택합니다. 

   ![image](https://github.com/user-attachments/assets/90ac32cf-602c-464d-af7f-02841792e947)

   **단위**는 **메트릭**을 선택합니다.

   ![image](https://github.com/user-attachments/assets/345467c7-4c17-4647-997b-b519dace5fac)


7. **Respond to Copilot**을 클릭하고, 왼쪽 창에서 **매개 변수** - **+ 출력 추가**를 클릭하여 아래와 같이 입력합니다.

    ![image](https://github.com/user-attachments/assets/83694e26-d0b6-4ad2-b857-682e0abbe7fb)

    번개 아이콘을 클릭하고, **오늘 예보 가져오기** 옆의 **더보기**를 클릭하고, 아래로 스크롤한 후에 **본문**을 선택합니다.

    ![image](https://github.com/user-attachments/assets/fb68fdc1-d1e4-465a-bcfc-18e762b6ba18)


8. 오른쪽 상단 메뉴에서 **초안 저장**을 클릭하여 저장합니다. 
    




5. 토픽을 저장하고, 테스트 합니다.

    ```
    뉴욕의 날씨가 어떤가요?
    ```
    ```
    서울의 기온은 어떤가요?
    ```
    
    ![image](https://github.com/user-attachments/assets/13308826-d819-4371-bdd0-ea4622ba6e09)
